# HDD-AntiSleep-Script

# 机械硬盘防休眠批处理脚本

一个简单有效的批处理脚本，用于防止机械硬盘（HDD）进入休眠状态，特别适合需要保持硬盘持续运行的环境。

## 功能特点

- ✅ 定期激活机械硬盘，防止休眠
- ✅ 轻量级，几乎不占用系统资源
- ✅ 可自定义检查间隔时间
- ✅ 可指定目标硬盘/分区
- ✅ 简单易懂的批处理代码

## 使用场景

1. **多硬盘系统** - 防止次要硬盘因长时间不访问而休眠
2. **NAS/服务器** - 保持所有硬盘随时可用
3. **数据备份** - 避免备份时因硬盘休眠导致的延迟
4. **老旧硬盘/硬盘盒场景** - 减少频繁启停对硬盘的损耗

## 使用方法

### 基础使用
1. 下载 `hdd-anti-sleep.bat`
2. 右键以管理员身份运行
3. 脚本会每30秒激活一次D盘

## 注意事项

⚠️ **重要提示**
1. 建议以管理员身份运行
2. 确保目标路径有写入权限
3. 长期使用可能略微增加硬盘磨损
4. 笔记本用户注意电源管理设置
5. SSD不需要此脚本，请勿对SSD使用

## 自定义配置

### 更改目标驱动器
将脚本中所有 `D:\` 替换为您想要防止休眠的驱动器盘符，例如：
- `C:\` 系统盘
- `E:\` 数据盘
- `F:\` 外接硬盘

### 调整时间间隔
修改 `timeout /t 30` 中的数字：
- `60` = 每分钟检查一次
- `300` = 每5分钟检查一次
- `600` = 每10分钟检查一次

## 高级用法

### 同时监控多个硬盘
创建多个版本的脚本，或修改脚本同时写入多个路径。

### 开机自启动
1. 按 `Win + R` 输入 `shell:startup`
2. 将脚本快捷方式放入启动文件夹
3. 在快捷方式属性中设置为"以管理员身份运行"

## 常见问题

**Q: 这个脚本安全吗？**
A: 是的，它只创建和删除临时文本文件，不会影响系统文件。

**Q: 会影响硬盘寿命吗？**
A: 对机械硬盘来说，频繁启停比持续运行更伤硬盘。但长期不间断运行也会产生正常磨损。

**Q: 如何停止脚本？**
A: 按 `Ctrl + C` 然后输入 `Y` 确认终止。

**Q: 需要一直开着命令行窗口吗？**
A: 是的，可以最小化运行。
